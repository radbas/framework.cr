# Framework.cr

The crystal web framework.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     framework:
       github: radbas/framework.cr
   ```

2. Run `shards install`

## Usage

```crystal
require "radbas-framework"
```
